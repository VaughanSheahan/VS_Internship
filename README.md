# VS_Internship
This repository is an ever-growing portfolio showcasing skills in many areas of computer science. It currently has few projects as it is in the early stage of it's development. Plans for expansion with a predicted timeline are as follows:  
  
  -Displaying ability with Java by coding implementations of well-known algorithms and data structures. (In progress: finished before 05/04)  
  -Using Python to demonstrate an understanding of simple data science and machine learning principles. (Planned: finished before 20/04)  
  -Making a version of the classic game 'snake' in C, guided by an online compilation of suggested projects in C. (Planned: finished early May)  
  -Constructing a real-time chatting application with public key encryption based on Node.js. (Planned: finished before the end of May)  
  
There are also plans to work on smaller projects such as calculators, Fizzbuzz, and other projects that show skill with niche aspects of coding.  
All programs will have both  a file where the code behind a program may be examined, and a compiled version that can be run in the terminal (used with git bash) or as an application.
